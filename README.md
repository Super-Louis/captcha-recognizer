# captcha-recognizer
## Features :smile:
- **可用于识别由字母及数字组合的图片验证码**
- **支持带标注验证码的生成，解决人工标注的困扰**  
- **利用opencv识别轮廓，对验证码进行切割**
- **random_forest及cnn模型在测试集上的预测精度均为99%**
## Problems :worried:
- **对于拉勾验证码识别率较低**
- **自动生成的验证码不能很好的模拟真实情况**
- **建议采用人工标注的真实数据**
